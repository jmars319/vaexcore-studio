# Studio Designer Shortcuts

This is the external shortcut reference for the Studio Scene Designer.

| Action | Default Shortcut |
| --- | --- |
| Save scene collection | Cmd/Ctrl + S |
| Undo | Cmd/Ctrl + Z |
| Redo | Cmd/Ctrl + Shift + Z or Cmd/Ctrl + Y |
| Copy selection | Cmd/Ctrl + C |
| Paste sources | Cmd/Ctrl + V |
| Select all stack sources | Cmd/Ctrl + A |
| Duplicate selection | Cmd/Ctrl + D |
| Delete selection | Delete or Backspace |
| Clear selection | Escape |
| Group selection | Cmd/Ctrl + G |
| Ungroup selection | Cmd/Ctrl + Shift + G |
| Toggle visibility | Cmd/Ctrl + Shift + H |
| Toggle lock | Cmd/Ctrl + L |
| Bring to front | Cmd/Ctrl + ] |
| Send to back | Cmd/Ctrl + [ |
| Nudge selection | Arrow keys |
| Coarse nudge | Shift + Arrow keys |
| Rotate left | [ |
| Rotate right | ] |

The in-app shortcut panel can rebind these locally. This file is a committed
reference for validation, release notes, and support docs.
