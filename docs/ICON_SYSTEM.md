# VaexCore Icon System

VaexCore uses responsive app icons.

- Large brand art can include the detailed neon scene and wordmark.
- Small app icon slots should use the simplified `V` mark, one app glyph, no text, and minimal background detail.
- Studio's small icon glyph is a streaming and recording control panel.

For macOS, `apps/desktop/src-tauri/icons/icon.icns` uses simplified artwork through 256 px and the detailed Studio brand artwork for 512 px and 1024 px slots.
